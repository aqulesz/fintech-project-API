# Fintech Backend API 💳

Digital wallet backend built with Spring Boot.

## 🚀 Tech Stack
- Java 17
- Spring Boot
- Spring Security
- JPA / Hibernate
- PostgreSQL
- JWT Authentication
- Docker

## 📌 Features
- User registration & login
- Secure authentication with JWT
- Wallet management
- Money transfers between users
- Transaction history
- Role-based authorization

## 🏗 Architecture
Layered architecture:
- Controller
- Service
- Repository

## 🔐 Security
- Password hashing (BCrypt)
- JWT authentication
- Input validation
- Exception handling

## 🐳 Run locally

mvn clean install
mvn spring-boot:run
